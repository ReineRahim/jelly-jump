# Jelly Jump 🪼

Jelly Jump is a simple arcade-style game built with **Python and
Pygame**.\
Players control a jellyfish that jumps through obstacles, collects
coins, and tries to survive as long as possible while the difficulty
increases.

## Features

-   Obstacle avoidance gameplay
-   Coin collection system
-   Multiple lives
-   Increasing difficulty levels
-   Sound effects and background music

## Controls

  Key     Action
  ------- -------------------------
  SPACE   Jump
  R       Restart after game over

## Technologies Used

-   Python
-   Pygame

## How to Run

1.  Install pygame:
```
    pip install pygame
```
2.  Run the game:

```
    python jellyjump.py
```
